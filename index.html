Skill Matrix App - User Manual

A step-by-step guide to using the JK Tech Skill Matrix application

Overview

The Skill Matrix App helps employees log, evaluate, and manage their technical and professional skills. Use this manual to learn how to log in, select skills, assign levels, and upload certifications.

Usage Steps

Step 1: Login

Visit the Skill App link: https://skill-matrix-140475459295.asia-south1.run.app/

Step 2: Post Login

You’ll be directed to the landing page shown below (the app home screen).

Step 3: Sign in with Microsoft

Click on “Sign in with Microsoft” and log in using your JK Tech official email ID.

Step 4: Continue to Skill Matrix

Once you’ve signed in successfully, you’ll see the Skill Matrix Welcome Page. Click on “Continue to Skill Matrix” to proceed to the main dashboard.

Step 5: Select Skills

On the Skill Matrix dashboard, you can select your skills under different Domains by clicking the domain name.

Alternatively, use the Search Bar to quickly locate a specific skill.

Step 6: Choose Your Level

Select your proficiency level for each skill.

Hover over the level number or click the question mark (?) icon to read the description for each level.

Step 7: View or Download Skill Matrix

Click the “Skill Matrix” tab to download and view the Excel file containing skills and domain data.

Step 8: Save Skill Selections

After selecting your levels for each skill, click on Save to record your progress.

Note: If you select the Expert level for any skill, you must enter your manager’s email for approval before saving.

Step 9: Add Certificates

Click on “Add Certificates” to open the certification section.

Step 10: Save Certificates

Choose your certificates from the available options, and click Save to finalize.
